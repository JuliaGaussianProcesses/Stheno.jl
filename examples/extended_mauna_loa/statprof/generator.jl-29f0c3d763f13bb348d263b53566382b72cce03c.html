65 """
66 IteratorSize(itertype::Type) -> IteratorSize
67
68 Given the type of an iterator, return one of the following values:
69
70 * `SizeUnknown()` if the length (number of elements) cannot be determined in advance.
71 * `HasLength()` if there is a fixed, finite length.
72 * `HasShape{N}()` if there is a known length plus a notion of multidimensional shape (as for an array).
73 In this case `N` should give the number of dimensions, and the [`axes`](@ref) function is valid
74 for the iterator.
75 * `IsInfinite()` if the iterator yields values forever.
76
77 The default value (for iterators that do not define this function) is `HasLength()`.
78 This means that most iterators are assumed to implement [`length`](@ref).
79
80 This trait is generally used to select between algorithms that pre-allocate space for their
81 result, and algorithms that resize their result incrementally.
82
83 ```jldoctest
84 julia> Base.IteratorSize(1:5)
85 Base.HasShape{1}()
86
87 julia> Base.IteratorSize((2,3))
88 Base.HasLength()
89 ```
